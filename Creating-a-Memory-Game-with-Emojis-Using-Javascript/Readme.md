# 🧠 Memory Game

**Memory Game** is a fun and engaging web-based memory game developed using JavaScript, HTML, and CSS. The objective is to match pairs of emojis within a grid, enhancing your memory and concentration skills. 🕹️✨

## 🌟 Features

- **📱 Responsive Design**: Ensures the game is playable on various devices.
- **🎨 Intuitive Interface**: Styled with `main.css` for an engaging user experience.
- **⚙️ Game Logic**: Managed by `engine.js` to handle game state and interactions.
- **🔄 Reset Functionality**: Allows players to restart the game via the reset button in `index.html`.

## 📂 Project Structure

- **`index.html`**: The main HTML file setting up the game layout.
- **`reset.css`**: CSS reset to ensure consistency across browsers.
- **`main.css`**: Main stylesheet for styling the game elements.
- **`engine.js`**: JavaScript file containing the game logic.
- **`.idea`**: IDE configuration files.

## 🎮 How to Play

1. Open the game in your web browser.
2. Click on a card to reveal the emoji.
3. Try to find and match its pair by remembering the card positions.
4. Repeat until all pairs are matched! 🌟

Have fun and sharpen your memory! 😊🃏
